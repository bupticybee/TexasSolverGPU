# TexasSolver GPU Distribution Notice

This repository is a public distribution repository for TexasSolver GPU release materials.

## What this repository includes

- release documentation
- release metadata
- Windows release packaging helpers
- the public `viewer` utility
- static public assets distributed with releases

## What this repository does not include

- the private `gpu_solver` source repository
- internal solver implementation details
- internal build systems beyond minimal release packaging helpers
- private training, testing, or development tooling

## Rights and limitations

- Access to this repository does not grant access to the private main source repository.
- Unless separately agreed by the author, this repository should be treated as a binary distribution channel plus auxiliary public files.
- No claim is made here that the private solver source code is open source or publicly licensed.

## Third-party dependencies

Some release bundles may include components or rely on runtimes distributed under their own terms, including WebView2 Runtime and NVIDIA software. Users are responsible for complying with those third-party terms.
